How does it work?

Domainion.co uses the WhoIsXMLAPI. Once a random word or group of words are generated, Domainion builds a domain name, and checks the WhoIsXMLAPI to make sure the domain name is available. If it's not, Domainion will generate a new domain name and try again, until an available domain name is found.

If you, the user, inputs your own domain, Domainion will use the WhoIsXMLAPI to check it's availability. Domainion will then tell you whether the domain is available or not.
